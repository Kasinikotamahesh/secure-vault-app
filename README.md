# 🔐 SecureVault System

A modern **web-based secure file vault** that provides encryption, authentication, and secure file handling.

---

## 🚀 Features

* 🔑 Encryption & Decryption using passphrase
* 🔐 Secure file storage simulation
* 👤 Login system with demo users
* 🧠 2FA (Two-Factor Authentication simulation)
* 🌙 Dark Mode support
* 📂 File vault with status (Encrypted / Plaintext)
* ⬇️ File download functionality

---

## 🛠️ Tech Stack

* React.js
* JavaScript
* HTML & CSS
